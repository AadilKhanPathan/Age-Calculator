# 🎂 Age Calculator

A simple and responsive Age Calculator built with **React** that calculates a user's exact age from their date of birth.

## 🚀 Features

* 📅 Select your date of birth using a date picker.
* 🧮 Calculate your current age instantly.
* ⚡ Fast and responsive React application.
* 🎨 Clean and user-friendly interface.
* ⌨️ Supports calculating age by clicking the button (and Enter key if implemented).

## 📦 Installation

1. Clone the repository.

```bash
git clone https://github.com/aadilkhanpathan/age-calculator.git
```

2. Navigate to the project folder.

```bash
cd age-calculator
```

3. Install dependencies.

```bash
npm install
```

4. Start the development server.

```bash
npm run dev
```

5. Open your browser and visit:

```
http://localhost:5173
```

## 🏗️ Build for Production

```bash
npm run build
```
